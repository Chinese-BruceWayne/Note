# BASIC

条件概率：

- $${\displaystyle P(A \mid B) = \frac{P(AB)}{P(B)} }$$

相互独立：

- 随机事件 A 与随机事件 B 相互独立 $$\Leftrightarrow P(AB) = P(A)P(B)$$
